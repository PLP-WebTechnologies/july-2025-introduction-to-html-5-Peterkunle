Introduction

Welcome! This page is a beginner-friendly guide to help you start playing the keyboard. No experience is needed.

Getting Started

1. Understand the Keys

The keyboard contains both white and black keys. The white keys are named A to G. Start by locating Middle C in the center.

2. Finger Numbers

Fingers are numbered 1 (thumb) to 5 (pinky). Practice using fingers 1-5 for basic notes with curved, relaxed posture.

3. Play a Simple Scale

Start with C Major: C-D-E-F-G. Use fingers 1-2-3-4-5 with your right hand. Practice slowly and focus on clarity.

Practice Tips

  • Practice 10–15 minutes daily
  • Use separate hands before combining
  • Keep wrists level and fingers relaxed
  • Start with familiar songs like “Twinkle Twinkle Little Star”
